# Ateneo Alumni Hub — Progressive Web Application (PWA)

This is the PWA-ready version of the Ateneo Alumni Hub, built to provide Ateneo de Davao University alumni with offline access, installable functionality, and improved performance.

## Features
- ✅ Installable via “Add to Home Screen”
- ✅ Works offline (offline.html fallback)
- ✅ Service worker caching
- ✅ Uses Ateneo branding
- ✅ Tested on Chrome & Safari mobile
